# PratsLab - A Web-Based Code Playground

PratsLab is a clean, focused, and feature-rich code playground that runs entirely in your browser. Built with a modern VS Code-style aesthetic, it provides a seamless environment for writing and testing both JavaScript and Python code without any backend setup. It saves your work locally and comes packed with professional themes and a flexible layout to match your workflow.

## üöÄ How to Use

Since PratsLab is a single HTML file with no server-side dependencies, getting started is incredibly simple:

1. **Download the File**: Save the `index.html` file to your local machine.  
2. **Open in Browser**: Double-click the file to open it in your favorite web browser (Chrome, Firefox, or Edge).  
3. **Start Coding**:
   - Use the **VS Code-style Explorer** on the left to manage your files and folders  
   - Right-click anywhere to create new files or folders  
   - Write your JavaScript or Python code in the Monaco editor  
   - Press **Run Code** or use `Ctrl+Enter` to execute  
   - View output in the multi-tabbed terminal on the right
